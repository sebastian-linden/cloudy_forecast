# Installation

## Stable release

To install Cloud Forecast, run this command in your terminal:

```sh
uv add cloud_forecast
```

Or if you prefer to use `pip`:

```sh
pip install cloud_forecast
```

## From source

The source files for Cloud Forecast can be downloaded from the [Github repo](https://github.com/sebastian-linden/cloud_forecast).

You can either clone the public repository:

```sh
git clone https://github.com/sebastian-linden/cloud_forecast
```

Or download the [tarball](https://github.com/sebastian-linden/cloud_forecast/tarball/main):

```sh
curl -OJL https://github.com/sebastian-linden/cloud_forecast/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd cloud_forecast
uv sync
```
