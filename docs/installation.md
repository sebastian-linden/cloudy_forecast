# Installation

## Stable release

To install Cloudy Forecast, run this command in your terminal:

```sh
uv add cloudy_forecast
```

Or if you prefer to use `pip`:

```sh
pip install cloudy_forecast
```

## From source

The source files for Cloudy Forecast can be downloaded from the [Github repo](https://github.com/sebastian-linden/cloudy_forecast).

You can either clone the public repository:

```sh
git clone https://github.com/sebastian-linden/cloudy_forecast
```

Or download the [tarball](https://github.com/sebastian-linden/cloudy_forecast/tarball/main):

```sh
curl -OJL https://github.com/sebastian-linden/cloudy_forecast/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd cloudy_forecast
uv sync
```
